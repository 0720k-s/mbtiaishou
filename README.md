このリポジトリは「MBTI診断結果に基づいて、相性の良いタイプを自動で探し、DMでリストアップしてくれるDiscord Bot」です。
独学でPython・AI・Discord APIを学びながら自作した実践プロジェクトです。

特徴

サーバー内のMBTIロール・履歴から自分のタイプを自動判定

相性の良いタイプ上位3つをDMでジャンプリンク付きでお知らせ

VC（ボイスチャット）入室時にプロフィールを自動表示

/diagnosis コマンドで診断パネルを呼び出し、ボタン一発診断

OpenAI・Gemini連携（機能拡張用/未使用部分あり）

PostgreSQL（asyncpg）で履歴やDB管理も可能（このサンプルではDBは最低限）
